
# ToDo app



## Tech Stack

**Language:** Dart

**Framework:** Flutter

**Backend:** Cloud Firestore

## Run Locally

Clone app

```bash
git clone https:github.com/hasti1109/todo-flutter
```

Navigate to project directory

```bash
cd todo-flutter
```

Get dependencies

```bash
flutter pub get
```

Run app locally

```bash
flutter run --debug
```
